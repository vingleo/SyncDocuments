----------------------------------------
不同 wavestate 模型的参考
本手册适用于所有 wavestate 模型，包括原始 wavestate、wavestate SE、wavestate mkII 和 wavestate 模块，统称为“wavestate”。

----------------------------------------
About the wavestate Editor/Librarian
The wavestate Editor/Librarian lets you organize and edit data on a wavestate connected to a Mac or Windows computer via USB, and transfer sound data back and forth between the wavestate and the computer.
For instance, you can:
• Edit Set Lists, Performances, Programs, Wave Sequences, and Wave Sequence Lanes, and Scales
• See the progress of sequences and the results of modulation in real-time
• Back up and restore all of the wavestate’s data
• Use Import and Export to transfer sounds to and from your computer, from one wavestate to another, and to and from the wavestate native plug-in
• Edit metadata for user sounds (Set Lists, Performances, Programs, Wave Sequences, Wave Sequence Lanes, and Scales), including the Name, Categories, and Collections (as shown on the wavestate’s display), as well as additional metadata including the Author and per-sound notes

----------------------------------------
关于 wavestate 编辑器/Librarian
wavestate 编辑器/Librarian 可让您通过 USB 组织和编辑连接到 Mac 或 Windows 计算机的 wavestate 上的数据，并在 wavestate 和计算机之间来回传输声音数据。
例如，您可以：
• 编辑设置列表、性能、程序、波形序列、波形序列通道和音阶
• 实时查看序列的进度和调制的结果
• 备份和恢复 wavestate 的所有数据
• 使用导入和导出在计算机之间、从一个 wavestate 到另一个 wavestate 以及 wavestate 本机插件之间传输声音
• 编辑用户声音的元数据（设置列表、性能、程序、波形序列、波形序列通道和音阶），包括名称、类别和集合（如 wavestate 的显示屏上所示），以及其他元数据，包括作者和每个声音的注释

####
Think of the Editor/Librarian as an external display for the wavestate

! Important: It’s best to think of the Editor/Librarian as an external display for the wavestate, instead of as a separate program. Everything that you see in the Editor/Librarian’s main and Set List windows is actually stored on the connected wavestate.

When you make edits or import data with the Editor/Librarian, you are directly editing the data on the connected wavestate. The Editor/Librarian can export data to your computer (for backup, or for transferring to another wavestate), but it has no data of its own. This also means that the Editor/Librarian cannot be used “off-line;” it only functions when connected to a wavestate.

####
将编辑器/库管理器视为wavestate的外部显示器

！重要提示：最好将编辑器/库管理器视为wavestate的外部显示器，而不是单独的程序。您在编辑器/库管理器的主窗口和设置列表窗口中看到的所有内容实际上都存储在连接的wavestate中。

当您使用编辑器/库管理器进行编辑或导入数据时，您直接在连接的wavestate上编辑数据。编辑器/库管理器可以将数据导出到您的计算机（用于备份或传输到另一个wavestate），但它没有自己的数据。这也意味着编辑器/库管理器不能“离线”使用；它只有在连接到wavestate时才能工作。

Do not operate your wavestate while data is being transmitted.

				Installation
----------------------------------------
wavestate software version 3.1.5 or later is required
Check the software version of the wavestate:
1. Press UTILITY and then SHIFT + < (PAGE -).

This will take you to the About page. The version number is shown at the top of the screen. If the version is earlier than 3.1.5, you’ll need to update the wavestate before continuing. The latest wavestate software can be downloaded from the Korg website (http://www.korg.com/ ).

####
What’s installed on my computer, and why?

The Editor/Librarian communicates with the wavestate using networking over USB, as opposed to MIDI. To automatically identify and connect with the wavestate over the network, the Editor/Librarian uses the mDNS standard.

On Windows, this requires Apple Inc.’s Bonjour, and so this is installed along with the Editor/Librarian application.

####
USB Network must be set to NCM
To use the Editor/Librarian, the wavestate’s USB Network parameter must be set to NCM. To make sure that this is set correctly:
1. Press UTILITY twice, and then hold SHIFT and press > (PAGE +).
This takes you to the MIDI & USB page, as shown below.
2. Make sure that USB Network is set to NCM.
3. If you had to change this from RNDIS to NCM, turn off and restart the instrument before proceeding.
USB Network
[RNDIS, NCM]
RNDIS: This is the legacy RNDIS protocol, which is no longer supported by the Editor/Librarian.
NCM: This is the Network Control Model protocol, for use with MacOS and Windows 10 and later.

！ Important: Changes to this parameter take effect only after the wavestate has been turned off and restarted.

----------------------------------------
MacOS
To install the software on MacOS:
1. On the wavestate, make sure that the USB Network parameter is set to NCM.
For more information, see “USB Network must be set to NCM” on page 4.

2. Connect your wavestate to the computer via USB.

3. Open the wavestate Editor/Librarian [version number] .dmg file in the Finder.
The disk image will open.

4. On the disk image, open the wavestate Editor/Librarian [version number] .pkg file.
The installer will start.

5. Follow the on-screen instructions to complete the installation. Enter your administrator password when prompted.
The installation will run, and the Editor/Librarian application will be installed in Applications/KORG/wavestate.

----------------------------------------
Windows
To install the software on Windows 10 or later (earlier versions are not supported):
1. On the wavestate, make sure that the USB Network parameter is set to NCM.
For more information, see “USB Network must be set to NCM” on page 4.

2. Double click “Korg wavestate Editor/Librarian [version number] Installer.exe.” The installer will start.

3. Follow the on screen instructions to complete the installation.
The installation will run, including the Editor/Librarian application, Apple Inc.’s Bonjour, and NCM networking support (using standard Windows drivers).

4. Connect your wavestate to the computer via USB.

In some cases, the wavestates may not be immediately recognized after installation. If this occurs, simply restart your computer.

####
Using the Editor/Librarian with two or more wavestates
！ Important: Before connecting multiple wavestates, make sure to set each instrument’s System ID to a different number.

To set the wavestate’s System ID:
1. Press the UTILITY button to go to the System Setup page.
If a different page appears at first, press UTILITY again.
2. Set the System ID parameter, at the bottom of the page, as desired.
The specific number isn’t important; just make sure that all connected wavestates have different System IDs.
3. In the wavestate Editor/Librarian, use the Devices menu to select the desired wavestate.
Note that data cannot be copied directly from one wavestate to another; instead, use Export to save the data from the first wavestate, and then Import to load the data into the second wavestate.

####
使用带有两个或更多wavestate的编辑器/库管理器
！重要提示：在连接多个wavestate之前，请确保将每个设备的系统 ID 设置为不同的数字。

要设置wavestate的系统 ID：
1. 按 UTILITY 按钮转到系统设置页面。
如果首先出现不同的页面，请再次按 UTILITY。
2. 根据需要设置页面底部的系统 ID 参数。
具体数字并不重要；只要确保所有连接的wavestate都有不同的系统 ID 即可。
3. 在wavestate编辑器/库管理器中，使用设备菜单选择所需的wavestate。
请注意，数据不能直接从一个wavestate复制到另一个wavestate；而是使用导出保存第一个wavestate的数据，然后使用导入将数据加载到第二个wavestate中。

----------------------------------------
Two modes: Editor and Librarian
The buttons in the upper right of the main window select whether you’re working with the Editor or the Librarian. For the most part, you can simply switch between the two without thinking about it. It’s important to note, however, that some menu commands may be available only in one mode, and that undo is handled separately for each mode.

----------------------------------------
两种模式：编辑器和图书管理员
主窗口右上角的按钮选择您是使用编辑器还是图书管理员。大多数情况下，您可以毫不犹豫地在两者之间切换。但需要注意的是，某些菜单命令可能仅在一种模式下可用，并且撤消操作针对每种模式单独处理。

####
Menu commands
In general, menu commands for file operations, such as importing, exporting, backing up, and restoring, are available only when the Librarian is active. Similarly, menu commands for saving Performances are available only when the Editor is active.

####
菜单命令
一般来说，文件操作的菜单命令（如导入、导出、备份和恢复）仅在 Librarian 处于活动状态时可用。同样，保存表演的菜单命令仅在 Editor 处于活动状态时可用。

####
Undo/Redo
The wavestate Editor/Librarian supports multiple levels of undo and redo for most actions, including importing data, deleting, renaming, editing Set Lists, editing parameters, and so on. For instance, you could import a bundle file containing a thousand objects, and then rename all of your Programs, and safely undo both actions in turn. Note that restoring from a backup is not undoable.

Undo/redo buffers are maintained separately for the Librarian and Editor modes. You’ll see this in the menu commands, whose names change based on the mode to either Librarian Undo/Redo or Editor Undo/Redo, respectively.

####
撤消/重做
wavestate 编辑器/库管理器支持大多数操作的多个级别的撤消和重做，包括导入数据、删除、重命名、编辑设置列表、编辑参数等。例如，您可以导入包含一千个对象的捆绑文件，然后重命名所有程序，并依次安全地撤消这两个操作。请注意，从备份中恢复是不可撤消的。

撤消/重做缓冲区是为库管理器和编辑器模式单独维护的。您将在菜单命令中看到这一点，其名称根据模式分别更改为库管理器撤消/重做或编辑器撤消/重做。

Basic use
Note: This manual describes how to use the Editor application. For details about how to use the wavestate, and how its functions and parameters work, please see the wavestate Operation Guide.

----------------------------------------
基本使用
注意：本手册介绍如何使用编辑器应用程序。有关如何使用 wavestate 以及其功能和参数如何工作的详细信息，请参阅 wavestate 操作指南。

####
Knobs and sliders
To edit knob values, drag vertically. To edit slider values, drag in the direction of the slider.
Most knobs and sliders can also be edited by hovering the cursor over the control, and then using the mouse wheel or dragging on the trackpad. The exception is when the controls are in a scrolling list, such as the detailed effects edit panels. In these lists, the mouse wheel and trackpad drag are used for scrolling, and so they are disabled for editing (to avoid unintended changes).
Double-click knobs and sliders to center them.

####
旋钮和滑块
要编辑旋钮值，请垂直拖动。要编辑滑块值，请沿滑块方向拖动。
大多数旋钮和滑块也可以通过将光标悬停在控件上，然后使用鼠标滚轮或在触控板上拖动来编辑。例外情况是控件位于滚动列表中，例如详细效果编辑面板。在这些列表中，鼠标滚轮和触控板拖动用于滚动，因此它们无法编辑（以避免意外更改）。
双击旋钮和滑块以使其居中。

####
Graphic editing
Envelopes, LFOs, Key Zones, Key & Velocity Zones, and Master EQ can be edited directly in their respective graphics.
To do this:
1. Hover the cursor over the graphic to show a colored handle (either a dot or a line).
2. Drag the handle to edit the value.
For Envelopes, there are handles for Curve in the middle of the A/D/R segments.
For Key & Velocity Zones, option-drag to edit the fade zone.

####
图形编辑
可以直接在各自的图形中编辑包络、LFO、键盘区、键盘和力度区以及主均衡器。
操作方法：
1. 将光标悬停在图形上以显示彩色手柄（点或线）。
2. 拖动手柄以编辑值。
对于包络，A/D/R 段中间有曲线手柄。
对于键盘和力度区，按住 Option 键拖动以编辑淡入淡出区。

wavestate keeps track of sounds, and some individual sound elements, using a database. This includes Performances, Programs, Wave Sequences, Wave Sequence Lanes, Effects Presets, Scales, Set Lists, and Multisamples. In the Editor/ Librarian, these appear as Selectors: 
Selectors

如果由于所选的类别、集合和搜索文本而隐藏了某些项目，则筛选列表图标会出现在 < 和 > 箭头之间。要清除过滤器并显示列表中的所有项目，请单击筛选列表图标。或者，打开声音浏览器并根据需要调整过滤器。有关更多信息，请参阅第 8 页上的“声音浏览器”。

Sound Browser

####
Overview
The Sound Browser is used for selecting any type of sound data, such as Performances, Programs, Wave Sequences, Wave Sequence Lanes, Multisamples, etc.

Filters Panel
The selections here help you narrow down the number of items in the Data List. Set the Search, Categories, and/or Collections as desired. The Filters Panel can be resized by dragging its right edge, to show one, two, or three columns of Categories and Collections.
Important: The Search, Categories, and Collections settings continue to affect data selection, even after the Sound Browser is closed. Each individual sound selector remembers these settings for as long as wavestate is open and unless a new parent sound is selected (for example, the Program is the parent sound of the Wave Sequence). For more information, see “Selectors” on page 7.

####
概述
声音浏览器用于选择任何类型的声音数据，例如表演、程序、波形序列、波形序列通道、多重采样等。

过滤器面板
此处的选择可帮助您缩小数据列表中的项目数量。根据需要设置搜索、类别和/或集合。可以通过拖动过滤器面板的右边缘来调整其大小，以显示一列、两列或三列类别和集合。

####
Data List
This shows the list of selectable sound data (Performances in the example above), as filtered by the Search, Category, and Collection settings in the Filters Panel. Click on an item in the list to select it for auditioning, or use the keyboard up/down arrows to browse through items one by one. Click in the list and type a few letters to select sounds by name.

数据列表
显示可选声音数据列表（上例中的演出），由“过滤器”面板中的“搜索”、“类别”和“集合”设置进行过滤。单击列表中的项目以选择它进行试听，或使用键盘的上下箭头逐个浏览项目。单击列表并输入几个字母以按名称选择声音。

Metadata columns
For each item, the list shows the Name, Collection, Category, Author, and Notes, as well as whether or not the item is locked factory data. You can drag the tops of the columns to re-arrange them, or to resize the columns.
Click on a column heading to sort; click again to reverse the sort order. The triangle icon shows which column is selected for sorting, and the direction of the triangle (up or down) shows the sort order.

####
元数据列
对于每个项目，列表显示名称、收藏、类别、作者和注释，以及该项目是否为锁定工厂数据。您可以拖动列的顶部以重新排列它们，或调整列的大小。
单击列标题进行排序；再次单击可反转排序顺序。三角形图标显示选择哪一列进行排序，三角形的方向（向上或向下）显示排序顺序。
	
####
Info Panel
This panel lets you view the metadata for the selected items, including the Name, Collection, Categories 1 & 2, Author, and Notes. The Inspector panel can be resized by dragging its left edge.

####
信息面板
此面板可让您查看所选项目的元数据，包括名称、收藏、类别 1 和 2、作者和注释。可以通过拖动检查器面板的左边缘来调整其大小。

####
OK/Cancel
Press OK to confirm the selection and close the window, or Cancel to revert to the previous selection.

####
确定/取消
按“确定”确认选择并关闭窗口，或按“取消”恢复到之前的选择。

####
Search
Type into this field to filter the list by searching for text in any of the metadata fields. Click on the “X” to clear the field.

####
搜索
在此字段中键入内容，通过搜索任何元数据字段中的文本来过滤列表。单击“X”可清除该字段。

####
Categories
Categories let you filter by the type of sound, such as basses, leads, bells, etc. Each sound can be assigned to two Categories, and each data type—Performances, Programs, etc.—has its own list of Categories. Click on a Category name to filter by that Category; click on the “X” to deselect all Categories.
When searching by Category, a sound will be shown if either of its Categories match the search criteria.

####
类别
类别可让您按声音类型进行筛选，例如低音、主音、铃铛等。每种声音可分配到两个类别，每种数据类型（表演、程序等）都有自己的类别列表。单击类别名称可按该类别进行筛选；单击“X”可取消选择所有类别。
按类别搜索时，如果任一类别符合搜索条件，则将显示声音。

Saving Sounds
The Performance, with its two Layers, is the main way of selecting, editing, and saving sounds. While you can save Programs, Wave Sequences, Wave Sequence Lane Presets, and Effects Presets, you don’t have to do so: all data is contained in the Performance.
Similarly, when you load any of these data types into a Performance, a new copy of the data is created in the Performance. Any edits affect only the local copy inside the Performance, and not the original data. This lets you edit freely without worrying about affecting other sounds.
To save a sound or preset:
1. Go to the Selector for the sound or preset. For more information, see “Selectors” on page 7.
2. Press the Save icon, or right-click/control-click (macOS) on the name to open the contextual menu and select the Save command.
The Save dialog will appear:
3. Set the Name, Author, Collection, and Categories as desired.
You can also edit all of this metadata later, using the Librarian window.

Important: changing the name does not automatically make a new copy of the sound! Always use Save As New when you want to make a copy.
4. Save the sound, using either Overwrite or Save As New.
To overwrite the existing sound, use Overwrite. To make a new copy and leave the existing sound unchanged, use Save As New. Factory sounds may be write-protected, in which case only “Save As New” is available.

####
Copy/Paste for Wave Sequence Steps
Wave Sequence Steps support Copy and Paste, as well as Cut, Insert Before, Insert After, and Delete. Shift-click to select a range of Steps, or select multiple non-continuous items by holding down the Command key on MacOS, or the Ctrl key in Windows. Once you’ve selected the desired step(s), right-click (or control-click on MacOS) to bring up the contextual menu and choose the desired copy/paste operation.

If you have cut or copied multiple steps, and then select a range of steps as the Paste destination, the following will happen:
• If a single step is selected, the steps will be pasted starting at that step, and then replace as many steps as necessary following that step.
• If you’ve selected exactly the same number of steps as are on the clipboard, even if they are discontiguous, Paste will replace only those selected steps.
• If you’ve selected fewer steps than are on the clipboard, even if they are discontiguous, Paste will replace the selected steps, and then replace as many steps as necessary following the last selected step.
• If you’ve selected more steps than are on the clipboard, Paste will replace the selected steps with a loop of the clip- board, stopping at the last selected step.

If you select a range of steps as the Insert Before or Insert After destination, only the first or last step matters. Insert Before refers to the first selected step, and Insert After refers to the last selected step.

If the Lane doesn’t already contain enough steps to complete the paste operation, more will be added to the end, up to the maximum of 64.

####
波形序列步骤的复制/粘贴
波形序列步骤支持复制和粘贴，以及剪切、在前插入、在后插入和删除。按住 Shift 键并单击可选择一系列步骤，或按住 MacOS 上的 Command 键或 Windows 上的 Ctrl 键可选择多个不连续的项目。选择所需的步骤后，右键单击（或在 MacOS 上按住 Control 键并单击）以调出上下文菜单并选择所需的复制/粘贴操作。

如果您剪切或复制了多个步骤，然后选择一系列步骤作为粘贴目标，则将发生以下情况：
• 如果选择了单个步骤，则将从该步骤开始粘贴步骤，然后在该步骤之后替换尽可能多的步骤。
• 如果您选择的步骤数与剪贴板上的步骤数完全相同，即使它们不连续，粘贴也只会替换那些选定的步骤。
• 如果您选择的步骤少于剪贴板上的步骤，即使这些步骤不连续，粘贴也会替换所选步骤，然后在最后选定的步骤之后替换尽可能多的步骤。
• 如果您选择的步骤多于剪贴板上的步骤，粘贴会用剪贴板的循环替换所选步骤，并在最后选定的步骤处停止。

如果您选择一系列步骤作为“插入之前”或“插入之后”目标，则只有第一步或最后一步很重要。插入之前是指第一个选定的步骤，插入之后是指最后一个选定的步骤。

如果通道中步骤不足以完成粘贴操作，则会在末尾添加更多步骤，最多为 64 个。

Step solo mode makes the Lane temporarily loop on the selected Step, for auditioning Multisamples or shapes, troubleshooting, etc.

####
UUIDs
The wavestate uses a database to keep sounds organized. Internally, sounds are identified not by their names, but rather by a unique tag attached to the file, called a UUID (“Universally Unique Identifier”). This means that even if a sound’s name has been changed, the system still knows it’s the same sound.

####
UUID
wavestate 使用数据库来保持声音的有序性。在内部，声音不是通过其名称来识别的，而是通过附加在文件上的唯一标签来识别的，称为 UUID（“通用唯一标识符”）。这意味着即使声音的名称已更改，系统仍然知道它是同一个声音。

When you write a sound from the wavestate front panel, “Overwrite” keeps the UUID the same, and “Save As New” creates a new UUID.

When you import data, the UUIDs in the sounds to be imported are compared with those already in the database. If a sound has the same UUID, but its contents are different, you’ll see the dialog described under “Data conflicts” on page 13.

当您从 wavestate 前面板写入声音时，“覆盖”会保持 UUID 不变，而“另存为新”会创建一个新的 UUID。

####
Scale settings
Scale Name
[Text]
This lets you edit the name of the Scale.
Important: Save always overwrites the existing Scale data, even if you change the name!
Type
[Octave Scale, Octave Scale, A=Master Tune, 128 Note Scale]
There are three supported scale types:
Octave Scale is a standard 12-note scale which repeats every octave.
Octave Scale, A=Master Tune is similar to the above, except that the Scale is automatically adjusted so that the pitch for A matches the Master Tune setting (e.g., A=440Hz), regardless of the Scale Key.
128 Note Scale allows separate tuning of each MIDI note, for Scales which don’t repeat on octave boundaries.
Cents
[–100…+100]
Each note can be detuned by up to 100 cents, flat or sharp.
Semitones
[–127…+127]
The Semitones parameter lets you detune a note by up to the entire MIDI range. As a simple example, to make the C key play a D pitch, set Semitones to +2. This is useful for scales which do not repeat on octave boundaries, or for repeated notes within an octave scale.

音阶设置
音阶名称
[文本]
这可让您编辑音阶的名称。
重要提示：即使您更改名称，“保存”也会始终覆盖现有音阶数据！
类型
[八度音阶、八度音阶、A=主调、128 音符音阶]
支持三种音阶类型：
八度音阶是标准的 12 音符音阶，每个八度重复一次。
八度音阶、A=主调与上述类似，不同之处在于音阶会自动调整，以便 A 的音高与主调设置相匹配（例如，A=440Hz），而与音阶键无关。
128 音符音阶允许对每个 MIDI 音符进行单独调音，适用于在八度边界上不重复的音阶。
音分
[–100…+100]
每个音符最多可以降调 100 音分，降调或升调。
半音
[–127…+127]
半音参数可让您将音符调低至整个 MIDI 范围。举个简单的例子，要使 C 键演奏 D 音高，请将半音设置为 +2。这对于不在八度边界上重复的音阶或八度音阶内的重复音符很有用。
