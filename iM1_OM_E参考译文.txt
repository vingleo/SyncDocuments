简介
感谢您购买 iM1 for iPad。为了充分享受您的 iM1 软件合成器，请仔细阅读本用户手册并将其放在手边以供将来参考。务必按说明使用本产品。

Main Features
One of the best-selling synthesizers in history, the Korg M1 defined the Workstation category for years to come. First introduced in 1988, this enormously successful PCM synthe- sizer was appreciated for its distinctive sonic character. With the iM1 for iPad, Korg has resurrected the legendary M1 as an up-to-date software synthesizer containing all of the orig- inal sounds, plus all-new sounds and new features as well.

主要特点
Korg M1 是历史上最畅销的合成器之一，它定义了未来几年的工作站类别。这款 PCM 合成器于 1988 年首次推出，取得了巨大成功，因其独特的声音特性而广受好评。借助适用于 iPad 的 iM1，Korg 将传奇的 M1 复活为一款最新的软件合成器，它包含所有原始声音以及全新的声音和新功能。

Over 2,700 Preset Sounds
As a workstation, the original M1 contained the vast selection of sounds needed to create an entire song-without the need for additional equipment.
超过 2,700 种预设音色
作为工作站，原始 M1 包含创作整首歌曲所需的大量音色，无需额外设备。

The iM1 for iPad contains all of the original sounds of the M1EX with its expanded internal PCM, plus all of the PCM data, Program data, and Combination data included in the 19 optional ROM cards, as well as the sound data of the Korg T1. (Some of this content requires a separate purchase.) A new "Best Program Card" that collects legendary M1 sounds is also featured. The iM1 for iPad contains all of the instrument, drum, and synthesizer PCM and Program data required for today's song-production, providing a staggering total of more than 2,700 preset sounds.
iPad 版 iM1 包含 M1EX 的所有原始音色及其扩展的内部 PCM，以及 19 个可选 ROM 卡中包含的所有 PCM 数据、程序数据和组合数据，以及 Korg T1 的声音数据。（部分内容需要单独购买。）还提供了一张新的“最佳程序卡”，其中收集了传奇的 M1 音色。iPad 版 iM1 包含当今歌曲制作所需的所有乐器、鼓和合成器 PCM 和程序数据，总共提供超过 2,700 种预设音色。

Easy-to-Use Interface
The iM1 for iPad interface has been designed to ensure maximum ease of use. The new Browser feature can quickly search for sounds by instrument type or tonal character, finding the right sound instantly-before your inspiration escapes!
Once the sound has been selected, it can be easily edited and customized using either the Easy page or the Performance page.
易于使用的界面
iM1 for iPad 界面的设计确保了最大程度的易用性。新的浏览器功能可以按乐器类型或音调特征快速搜索声音，在您的灵感消失之前立即找到合适的声音！
一旦选择了声音，就可以使用 Easy 页面或 Performance 页面轻松编辑和自定义它。

New Features for Enhanced Performance
The iM1 for iPad adds many new features that were wished-for by users of the original M1. New to the iM1 are Filter resonance, VDA modulation, and compressor effects. The two effects processors of the original M1 have been replaced by Two Insert effects for each Timbre (Track) and two Master effects, providing for richer sound creation. In addition, the all-new Multi mode allows the iM1 for iPad as to be used as an eight-part multi-timbral sound module.
增强性能的新功能
iPad 版 iM1 增加了许多原版 M1 用户所期望的新功能。iM1 的新功能包括滤波器共振、VDA 调制和压缩器效果。原版 M1 的两个效果处理器已被两个用于每个音色（音轨）的插入效果和两个主效果所取代，从而提供更丰富的声音创作。此外，全新的多模式允许 iPad 版 iM1 用作八声部多音色声音模块。

iM1 Modes and Structure
The iM1 for iPad loads the Program data independently for each mode; Programs within each mode can be edited with- out affecting the other modes.
iM1 模式和结构
iPad 版 iM1 会为每种模式独立加载程序数据；每种模式中的程序都可以编辑，而不会影响其他模式。

Combination (Combi) mode
In this mode, the sound consists of eight Timbres; each Timbre is individually assigned to its own Program.
组合 (Combi) 模式
在此模式下，声音由八种音色组成；每种音色都单独分配给自己的程序。

图略

Each Combination contains the following parameters:
• Individual parameters for each Timbre, such as the output level, pan, and Insert effect on/off settings.
• MIDI-related parameters for each Timbre.
• Master effect parameters.
• Sound-creation parameters for each Program.
Combinations can be selected from the 32 preset cards and from the new Combinations (the KLC PRESET card). Combination that you have tweaked, edited, or created from scratch can be saved to any of the six User cards. Each User card can hold 50 Combinations, for a total of 300 User Combinations. In addition, the iM1 provides four user cards that are saved in the session, and two library cards that are stored within iM1 itself.
每个组合包含以下参数：
• 每个音色的单独参数，例如输出电平、声像和插入效果开/关设置。
• 每个音色的 MIDI 相关参数。
• 主效果参数。
• 每个程序的声音创建参数。
可以从 32 个预设卡和新组合（KLC PRESET 卡）中选择组合。您调整、编辑或从头开始创建的组合可以保存到六张用户卡中的任何一张。每张用户卡可以容纳 50 个组合，总共 300 个用户组合。此外，iM1 提供四张保存在会话中的用户卡和两张存储在 iM1 本身内的库卡。

Multi mode
In this mode, the iM1 for iPad can be used as an eight-part multi-timbral sound module.
多模式
在此模式下，iPad 版 iM1 可用作八声部多音色声音模块。

Multi mode and Combination mode are structured almost
identically, but have the following differences.
• Default MIDI channel setting for each track (track 1 is channel 1, track 2 is channel 2, …)
• In Multi mode, the master effects are used as send effects.
The send levels, return levels, and return destinations will determine how the signal is routed. If you lower the send levels to 0, the signals from the insert effects will be output directly, bypassing the master effects. If you raise the send levels, the direct sound will be mixed with the signals from the master effects and then output.
多音模式和组合模式的结构几乎相同，但有以下区别。
• 每个音轨的默认 MIDI 通道设置（音轨 1 为通道 1，音轨 2 为通道 2，……）
• 在多音模式下，主效果用作发送效果。
发送级别、返回级别和返回目的地将决定信号的路由方式。如果将发送级别降低到 0，则插入效果的信号将直接输出，绕过主效果。如果提高发送级别，则直接声音将与主效果的信号混合，然后输出。

Except for the points listed above, Combination mode and Multi mode have the same structure. You can use Combina- tion mode as a multi-timbral sound module, or use Multi mode to create a single sound just like a combination.
除了上面列出的几点外，组合模式和多重模式具有相同的结构。您可以将组合模式用作多音色声音模块，或使用多重模式创建像组合一样的单一声音。

Program mode
This mode is used to play and edit individual Programs.
程序模式
此模式用于演奏和编辑单个程序。

Programs can be selected from the 32 preset cards and from the new KLC PRESET card and iM1 Special card. Programs that you have tweaked, edited, or created from scratch can be saved to any of the six User cards. Each User card can hold 50 Programs, for a total of 300 user Programs.
可以从 32 个预设卡以及新的 KLC PRESET 卡和 iM1 特殊卡中选择程序。您调整、编辑或从头创建的程序可以保存到六张用户卡中的任何一张。每张用户卡可以容纳 50 个程序，总共 300 个用户程序。

Each Program uses one of the three oscillator modes: Single, Double, or Drums.
每个程序使用三种振荡器模式之一：单振荡器、双振荡器或鼓振荡器。

Single
Single Programs use a single oscillator. The oscillator consists of the OSC, VDF, VDA, and Insert FX sections. The multisounds available for use by the oscillator can be selected from the 16 preset cards.
单路
单路程序使用单一振荡器。振荡器由 OSC、VDF、VDA 和 Insert FX 部分组成。振荡器可用的多音效可从 16 个预设卡中选择。

Double
Double Programs use two oscillators. Each oscillator consists of the OSC, VDF, and VDA sections. The combined output of the oscillators is routed to the Insert FX.
双路
双路程序使用两个振荡器。每个振荡器由 OSC、VDF 和 VDA 部分组成。振荡器的组合输出被路由到 Insert FX。

Drums
Drum Programs use a Drum Kit as the oscillator. The Drum Kit oscillator consists of an OSC, VDF, VDA, and the Insert FX.
鼓
鼓程序使用鼓组作为振荡器。鼓组振荡器由 OSC、VDF、VDA 和 Insert FX 组成。

On the original M1, the SETTINGS mode was used to edit a Drum Kit. The iM1 for iPad now allows the Drum Kit parameters to be part of the Program.
在原版 M1 上，SETTINGS 模式用于编辑鼓组。iPad 版 iM1 现在允许将鼓组参数作为程序的一部分。

Drum Kits can be selected from 21 of the preset cards, and from the new KLC PRESET card. Drum Kits that you have tweaked, edited, or created from scratch can be saved to any of the four User cards (2 User cards; 2 Library cards). Each User card can hold 20 Drum Kits, for a total of 80 User Drum Kits.
鼓组可以从 21 个预设卡和新的 KLC PRESET 卡中选择。您调整、编辑或从头创建的鼓组可以保存到四张用户卡中的任何一张（2 张用户卡；2 张库卡）。每张用户卡可容纳 20 个鼓组，总共 80 个用户鼓组。

SETTINGS mode
This mode is used to access and adjust parameters that affect the entire iM1 for iPad operation, such as Master Tune, Transpose, Global MIDI Channel, MIDI Filter, and User Scale.(Refer to "SETTINGS mode" on page 56)
设置模式
此模式用于访问和调整影响整个 iM1 for iPad 操作的参数，例如主调音、移调、全局 MIDI 通道、MIDI 滤波器和用户音阶。（请参阅第 56 页的“设置模式”）

Kaoss pads
In place of the keyboard, the iM1 can instead provide two Korg Kaoss pads as the control surface. By dragging across the Kaoss pad surface, you can play sounds and control mul-tiple parameters at once-those assigned to the X-axis (horizontal motion) and those assigned to the Y-axis (vertical motion). Although the two Kaoss pads look nearly identical, their functions are different. The left-hand pad (Kaoss Pad L) offers assignable parameters. The right-hand pad (Kaoss Pad R) is set to control the pitch and the inversion type when playing chords.

Using MIDI control change messages to control the speed of the Rotary Speaker When the Rotary Speaker has been selected as a Master effect, the rotation speed can be switched using MIDI CC (Control Change) messages.

Using MIDI CC messages to control the speed of the Rotary Speaker When the Rotary Speaker has been selected as an Insert effect, the rotation speed can be switched using MIDI CC (Control Change) messages.
Transmitting a CC#76 (Vibrato Rate) message from your external MIDI device on the MIDI Channel (Refer to "MIDI Ch" on page 24) with a value of 64-127 will switch the rotary speed to Fast; transmitting a value of 00-63 will switch the rotary speed to Slow.
使用 MIDI CC 消息控制旋转扬声器的速度 当选择旋转扬声器作为插入效果时，可以使用 MIDI CC（控制变化）消息切换旋转速度。
从 MIDI 通道（请参阅第 24 页的“MIDI Ch”）上的外部 MIDI 设备传输值为 64-127 的 CC#76（颤音速率）消息会将旋转速度切换为快速；传输值为 00-63 会将旋转速度切换为慢速。

################################################
				Writing, Saving, and Loading Data
Writing to internal memory
Any of the settings you edit or create can be stored in the iM1 for iPad's internal memory. On the original M1, the Programs used by a Combination used the same data as the Programs in Program mode. The iM1 for iPad loads Program data into the edit buffer independently for each mode (Combination, Multi, or Program), which means that editing a Program in Program mode will not affect the sound of Programs used in Combination mode or Multi mode.
By using the File menu, the data that you edit can also be saved for individual sections. Programs that are saved in library cards can be shared across the entire application.
The WRITE command does not save the data to your computer; it only stores it to the iM1 for iPad's internal memory. Any data you've stored will be lost if you exit the application without saving. If you want to keep your stored Combinations and Programs, click the [FILE] button and use "Save" to save them (“Session”).
Session
By saving a Session, the Combi and Program edit data-and the data that is saved in the user cards-is saved all together.
Session commands
• New: Resets all stored data of the session, and returns the software to its initial state.
• Open: Loads session data from a file.
• Save: Saves session data by overwriting the old file.
• Save As: Saves session data under a different name.
• Import from KLC: Imports KLC M1 compatible data.
• Export toKLC: Saves data in a form that is compatible with the KLC M1.
• Import from M1: Imports data created on the original M1 series as a system exclusive file into iM1 for iPad.
The data to be imported or the data that you've exported can be written to or read from your iPad via iTunes software on your computer.
When you import each type of data, it will overwrite the USER card. If you want to keep the current session, use "Save as" to back it up before importing.
写入、保存和加载数据
写入内部存储器
您编辑或创建的任何设置都可以存储在 iM1 for iPad 的内部存储器中。在原始 M1 上，组合使用的程序使用的数据与程序模式下的程序相同。iM1 for iPad 会针对每种模式（组合、多重或程序）将程序数据独立加载到编辑缓冲区中，这意味着在程序模式下编辑程序不会影响组合模式或多重模式下使用的程序的声音。
通过使用文件菜单，您还可以为各个部分保存您编辑的数据。保存在库卡中的程序可以在整个应用程序中共享。
WRITE 命令不会将数据保存到您的计算机；它只会将数据存储到 iM1 for iPad 的内部存储器中。如果您退出应用程序而不保存，您存储的任何数据都将丢失。如果您想保留存储的组合和程序，请单击 [FILE] 按钮并使用“保存”来保存它们（“会话”）。
会话
通过保存会话，Combi 和程序编辑数据以及保存在用户卡中的数据将全部保存。
会话命令
• 新建：重置会话的所有存储数据，并将软件返回到其初始状态。
• 打开：从文件加载会话数据。
• 保存：通过覆盖旧文件保存会话数据。
• 另存为：以不同的名称保存会话数据。
• 从 KLC 导入：导入 KLC M1 兼容数据。
• 导出到 KLC：以与 KLC M1 兼容的形式保存数据。
• 从 M1 导入：将原始 M1 系列上创建的数据作为系统专用文件导入 iM1 for iPad。
要导入的数据或您已导出的数据可以通过计算机上的 iTunes 软件写入或从 iPad 读取。
当您导入每种类型的数据时，它将覆盖用户卡。如果要保留当前会话，请在导入之前使用“另存为”进行备份。

Writing Procedure
Combination mode, Multi mode, Program mode:
1. Click the [WRITE] button located in the upper left, and choose either Selected Program or Program from the menu that appears. The WRITE PROGRAM dialog box will appear with the instrument and sound character that are specified for the Program.
You can also access the WRITE PROGRAM dialog box by selecting the Write menu command for each Timbre (Track).
2. Select the writing-destination card. Tap one of the [LIBRARY1], [LIBRARY2], or [USER1]--[USER4] buttons shown below.
3. If you want to change the name before saving, tap on the Program name displayed at the top of the window, and using the pop-up keyboard, type in a name and press the [Enter] key.
4. Select the type of instrument in the SEARCH "INSTRUMENTS" area, and the character of the sound in the "CHARACTER" section. You can select multiple items for the type of instrument or sound character. (Refer to "4. INSTRUMENTS" on page 15 and "5. CHARACTER" on page 15)
5. Specify a name for the Program that you want to save, and press the [Enter] button.
6. Close the keyboard.
The Program is written to the Write-Destination User Card.

组合模式和多模式的 Performance 页面以及程序模式的 Performance Edit 页面中的程序调整参数在写入时应用于程序参数，其值将重置为 00。

写入步骤
组合模式、多重模式、程序模式：

您也可以通过选择每个音色（音轨）的写入菜单命令来访问 WRITE PROGRAM 对话框。

Drum Kits
Drum kits you created or edited in Combination, Multi, or Program mode can be written (stored) to a user card in the iM1 for iPad.
The iM1 for iPad provides two internal “user cards” for storing drum kits. Since each card holds 20 drum kits, this gives you a total of 40 drum kits stored within the M1.
Settings that are remembered when you Write OSC page DRUM KIT EDIT section parameters other than KBD CHASE will be remembered when you Write.

Writing
1. Click the [WRITE] button in the OSC page DRUMS section of any mode.
The WRITE DRUM KIT dialog box will appear with the attribute that is specified for the Drum Kit selected..
2. Select the writing-destination card.
Tap one of the [LIBRARY1], [LIBRARY2], [USER1], or [USER2] buttons shown below.
3. Select the desired attribute(s) for this Drum Kit. You may select multiple attributes.
4. Specify a name for the Program that you want to save, and press the [Enter] button.
5. Click the [OK] button.
The Drum Kit will be written to the user card.
鼓组
您在组合、多重或程序模式下创建或编辑的鼓组可以写入（存储）到 iM1 for iPad 中的用户卡中。
iM1 for iPad 提供两个用于存储鼓组的内置“用户卡”。由于每张卡可容纳 20 个鼓组，因此 M1 中总共存储了 40 个鼓组。
写入时记住的设置（除 KBD CHASE 外）将在您写入时记住。

写入
1. 单击任何模式的 OSC 页面 DRUMS 部分中的 [WRITE] 按钮。
将出现 WRITE DRUM KIT 对话框，其中选定了为鼓组指定的属性。
2. 选择写入目标卡。
点击下面显示的 [LIBRARY1]、[LIBRARY2]、[USER1] 或 [USER2] 按钮之一。
3. 选择此鼓组所需的属性。您可以选择多个属性。
4. 为要保存的程序指定名称，然后按 [Enter] 按钮。
5. 单击 [OK] 按钮。
鼓组将写入用户卡。

Renaming
Using the keyboard that is displayed when Writing, enter a name and then press the Return key.
Names can be entered for a combination, Program, or Drum Kit that can be written to a memory card. Names cannot be entered for a multisound or drum sound.
重命名
使用写入时显示的键盘输入名称，然后按回车键。
可以为可写入存储卡的组合、程序或鼓组输入名称。无法为多音色或鼓音输入名称。
