SHIFT

ENTER for accelerated editing
Hold down ENTER to accelerate editing or scrolling. For example:
• Normally, the VALUE knob works as a “fine” value input when moved slowly, with “coarse” bigger jumps when moved quickly. To restrict it to “coarse” value changes, hold ENTER while turning VALUE.

ENTER 用于加速编辑

• 按住 ENTER 键并按下 < 或 >（或转动 VALUE 拨盘）可在列表中滚动一整页，例如在选择 Performances（性能）、Multisamples （多采样） 或 Mod （Mod） 路由时。

Modulation
Most front-panel knobs and on-screen parameters are modulatable. Hold MOD and press > to create a new modulation routing. When viewing and editing modulations, hold MOD and press < to filter the list. See the Mod List Info page for an on-screen list of shortcuts.
调制
大多数前面板旋钮和屏幕上的参数都是可调制的。按住 MOD 并按下 > 键以创建新的调制路由。在查看和编辑调制时，按住 MOD 并按下 < 键来过滤列表。参见 Mod List Info 页面以获取屏幕上的快捷方式列表。

Mod Knobs
You can use the Mod Knobs in real-time performance, and also save the results as new sounds. The names are suggestions; they will do different things depending on the specific sound. The values are stored, and can themselves be modulated. Each Layer’s Program has its own set of Mod Knobs, and an overall set of Performance Mod Knobs can control any or all Layers at once. Sometimes, one knob (such as 1/MASTER) may enable the functionality of other knobs.
Mod 旋钮
您可以在实时演奏中使用 Mod 旋钮，也可以将结果保存为新声音。这些名字是建议;它们会根据特定的声音做不同的事情。这些值被存储起来，并且本身可以被调制。每个 Layer 的 Program 都有自己的一组 Mod 旋钮，而一整套 Performance Mod 旋钮可以同时控制任何或所有 Layer。有时，一个旋钮（例如 1/MASTER）可能会启用其他旋钮的功能。

Help pages
Hold SHIFT and press the (Randomize) button to bring up a set of pages containing shortcuts and usage tips.
帮助页面
按住 Shift 并按下 （Randomize） 按钮可调出一组包含快捷方式和使用提示的页面。

Structure
The wavestate plays one Performance at a time. A Performance has four Layers, A…D, a Vector Envelope, and a master reverb and EQ. Each Layer contains an Arpeggiator, a Program, and other settings such as MIDI channel, key and velocity zones, etc. A Program includes a Wave Sequence or a single Multisample, a Filter, an Amp, a per-voice Vector Envelope, a set of modulators, a modulation matrix, and three effects: Pre FX, Mod FX, and Delay.
wavestate 一次演奏一个 Performance。一个Performance
 有四个音色层，A...D、一个 Vector 包络以及一个主混响和均衡器。每个音色层都包含一个琶音器、一个音色和其他设置，例如 MIDI 通道、键位和力度区域等。程序包括一个 波形音序器（Wave Sequence） 或单个多重采样（Multisample）、一个滤波器、一个 放大器、一个按每声部的向量包络、一组调制器、一个调制矩阵和三个效果器：Pre FX、Mod FX 和 Delay。

Using Set Lists
Set Lists let you group and order Performances for gigs or projects. A Set List has 64 Slots, arranged into four banks A- D, corresponding to MIDI Program Change messages 1-64. Note that Set Lists don’t contain separate copies of their sounds; they just point to Performances stored in the database.
To select sounds in the current Set List:
使用 Set List
Set List 可让您对演出或项目的演奏进行分组和排序。Set List 有 64 个插槽，排列成四个库 A-D，对应于 MIDI 程序更改消息 1-64。请注意，Set List 不包含其声音的单独副本；它们只是指向存储在数据库中的演奏。

Using Note Trigger (module only)
Note Trigger lets you play sounds using the 1-16 buttons as an alternative keyboard—convenient if you don’t have a full keyboard connected. Note Trigger is only available on the wavestate module.
To use Note Trigger:
1. Hold SHIFT and press WSEQ STEPS.
The Note Trigger page appears:
Buttons 1-13 illuminate to create a keyboard pattern from C to C, with dimmed lights showing the white keys.
2. To play notes, press buttons 1-13.
3. To change the octave, use buttons 15 (down) and 16 (up), or edit the Octave parameter on the display.
4. To set the velocity of the triggered notes, use the Velocity parameter.
5. To cancel Note Trigger, press either SET LIST or WSEQ STEPS.
While using Note Trigger, you can use almost all of the instrument as you would normally. There are a few exceptions:
• When selecting Performances, Programs, Wave Sequences, etc., the 1-16 buttons continue to function as Note Triggers, and so they are not available for selecting Categories.
• Entering a Wave Sequence editing page cancels Note Trigger, since the 1-16 buttons need to be used for Step selection.

				Write
Saving Sounds
The Performance, with its four Layers, is the main way of selecting, editing, and saving sounds. While you can save Programs, Wave Sequences, and Wave Sequence Lane Presets, you don’t have to do so: all data is contained in the Performance.

Similarly, when you load any of these data types into a Performance, a new copy of the data is created in the Performance. Any edits affect only the local copy inside the Performance, and not the original data. This lets you edit freely without worrying about affecting other sounds.

To save:

				Wave Sequencing 2.0

Overview
Wave Sequencing 2.0 is a dynamic system. Instead of each step being a matched set of sample, pitch, duration, etc., the different parameters are separated into “lanes.” Each lane can have a different number of steps, and its own start, end, loop start, and loop end.
These points (and other parameters in the lanes and their steps) can in turn be modulated on a per-note basis, using velocity, LFOs, envelopes, etc. Each time the sequence moves forward, values from the individual lanes are combined to create the output. The result can be either ever-changing rhythmic patterns or smooth, organic, crossfading timbres. Individual, modulatable step probabilities provide additional variation.
概述
波形序列 2.0 是一个动态系统。每个步骤不再是一组匹配的样本、音高、持续时间等，而是将不同的参数分成“通道”。每个通道可以有不同数量的步骤，以及自己的开始、结束、循环开始和循环结束。
这些点（以及通道及其步骤中的其他参数）可以依次根据每个音符进行调制，使用速度、LFO、包络等。每次序列向前移动时，来自各个通道的值都会组合起来以创建输出。结果可以是不断变化的节奏模式或平滑、有机、淡入淡出音色。单独的、可调节的步骤概率提供了额外的变化。

To understand how Random Order works, imagine that each Step is on a note-card. 要了解随机顺序的工作原理，请想象每个步骤都在一张记事卡上。每次循环播放时，或者每当起始步骤或结束步骤发生变化时，从起始步骤到结束步骤的记事卡都会被打乱并分配到新的数字（A1、A2 等）。如果在开始之前有任何步骤，它们将不包含在打乱中；相反，它们会留在原处并保留其原始值，即使它们会根据循环设置播放。结束步骤之后的步骤将被忽略。出于播放目的，开始、结束、循环开始和循环结束都适用于新分配的数字。与更改开始步骤或结束步骤不同，更改循环开始步骤或循环结束步骤不会导致重新洗牌。

Step Probability
Each individual Step has a Probability setting, from 0% to 100%. This controls how likely the Step is to play. As with most Step parameters, Probability can be modulated; for instance, moving a Mod Knob might make some Steps more likely to occur, and other steps less likely to occur.

If a step has less than 100% Probability and loses the roll of the dice, as it were, it is skipped. (See note about the Timing Lane, below.) Since the Wave Sequence is generated in real-time, we need to avoid a situation in which we repeatedly roll the dice, lose the throw, and roll again and again; eventually, we need to make some sound! So, if two steps in a row are skipped, the third step always is played—even if its probability is 0%.

Timing Lane probability The Timing Lane works slightly differently regarding probability: “skipped” steps are absorbed into the previous step.

For instance, if A1 and A2 are both 16th notes, and A2 is skipped due to probability, A1 becomes an 8th note. This maintains the overall length of the rhythm.
步骤概率
每个步骤都有一个概率设置，从 0% 到 100%。这控制步骤播放的可能性。与大多数步骤参数一样，概率可以调节；例如，移动调制旋钮可能会使某些步骤更有可能发生，而其他步骤不太可能发生。

如果某个步骤的概率低于 100%，并且掷骰子失败，则将其跳过。（请参阅下面有关计时通道的注释。）由于波形序列是实时生成的，我们需要避免反复掷骰子、掷骰子失败并反复掷骰子的情况；最终，我们需要发出一些声音！因此，如果跳过连续两个步骤，则第三个步骤始终会播放 - 即使其概率为 0%。

计时通道概率 计时通道在概率方面的工作方式略有不同：“跳过”的步骤会被吸收到上一步中。

例如，如果 A1 和 A2 都是十六分音符，由于概率原因 A2 被跳过，则 A1 变为八分音符。这样可以保持节奏的整体长度。

				Vector
Overview
Vector Synthesis lets you control the sound by moving the Vector Joystick, using the programmable Vector Envelope, or the combination of the two.
What does Vector mean?
Modulation generally works by moving a single control, like a slider, in a straight line. At one end of the control, the modulation is at its minimum; at the other end, it’s at its maximum.
Vector Synthesis is a little different. It works by moving around a point on a two-dimensional plane, both left-right and up-down.
You can think of this point as being positioned on two different lines at once: a left-right line (the X axis), and an up-down line (the Y axis).
In other words, instead of just having one value (like a slider), each Vector point has two values: one for X, and one for Y. See the graphic below for an example.
向量合成可让您通过移动向量操纵杆、使用可编程向量包络或两者结合来控制声音。
向量是什么意思？
调制通常通过沿直线移动单个控件（如滑块）来工作。在控件的一端，调制处于最小值；在另一端，调制处于最大值。
向量合成略有不同。它的工作原理是围绕二维平面上的一个点移动，左右和上下。
您可以将此点视为同时位于两条不同的线上：一条左右线（X 轴）和一条上下线（Y 轴）。
换句话说，每个向量点不只有一个值（如滑块），而是有两个值：一个用于 X，一个用于 Y。请参阅下图中的示例。

In addition to moving the point directly with the Vector Joystick, you can also use the Vector Envelope to move its position automatically over time, as shown below.
除了使用矢量操纵杆直接移动点之外，您还可以使用矢量包络随时间自动移动其位置，如下所示。

Vector Joystick and Vector Envelope

You can move the Vector point using the combination of the Vector Joystick and the Vector Envelope. The two work together, although you don’t have to use both at the same time.
When the Vector Joystick is in its center position, the Vector Envelope has complete control over the point’s position.
Likewise, when the Vector Envelope is in the center, the Vector Joystick has complete control.
您可以使用矢量操纵杆和矢量包络的组合来移动矢量点。两者协同工作，但您不必同时使用两者。
当矢量操纵杆处于其中心位置时，矢量包络可以完全控制该点的位置。
同样，当矢量包络位于中心时，矢量操纵杆可以完全控制。

When the Vector Envelope is in use, the Vector Joystick offsets the Vector Envelope position by up to halfway across either axis. For instance, if the Envelope is all the way to the right of the X axis, and the Joystick is all the way to the left, the actual Vector position will be in the center of the X axis.
使用矢量包络时，矢量操纵杆会将矢量包络位置偏移至任一轴的一半。例如，如果包络位于 X 轴的最右侧，而操纵杆位于最左侧，则实际矢量位置将位于 X 轴的中心。

Vector Volume Control and modulation
Vector does several things:
• It generates four per-voice modulation signals, Vector Env A/B/C/D
• It generates similar modulation signals at the Performance level, shared by all voices, for modulation of things like effects and the Performance Mod Knobs
• When Vector Volume is On, it also controls the relative volumes of Layers A-D on a per-voice basis
You can also use the Vector Joystick’s X- and Y-axes as modulation sources, independent from the Vector Envelope.
Since the Vector Envelope’s settings are stored in the Performance, it’s best used for Performance-specific purposes, such as changing parameters (e.g. filter cutoff) in conjunction with volume, or interactions between layers.

• 它在性能级别生成类似的调制信号，由所有声音共享，用于调制效果和性能调制旋钮等

Filter Type (FILTER TYPE) 
[ Polysix, MS-20 LP, MS-20 HP, 2-pole LP, 2-pole HP, 2-pole BP, 2-pole BR, 4-pole   LP, 4-pole HP, 4-pole BP, 4-pole BR, Multi Filter ] Pressing the front panel FILTER TYPE button selects and edits the Type parameter. Three useful settings are available directly: the Polysix LP, the MS-20 LP, and a 2-pole resonant HP. Many more types are available by editing the on- screen parameter. If in doing so you select one of the three types listed on the front panel, the front-panel FILTER TYPE LEDs will update to show the selection. Selecting any other Filter Type lights the MORE LED. If you switch between Types from the front-panel button, the MORE position remembers the last-used “more” Type.
LP (Low Pass). This cuts out the parts of the sound which are higher than the cutoff frequency. Low Pass is the most common type of filter, and is used to make bright timbres sound darker.
HP (High Pass). This cuts out the parts of the sound which are lower than the cutoff frequency. You can use this to make timbres sound thinner or more buzzy.
BP (Band Pass). This cuts out all parts of the sound, both highs and lows, except for the region around the cutoff frequency. Since this filter cuts out both high and low frequencies, its effect can change dramatically depending on the cutoff setting and the oscillator’s multisample.
With low resonance settings, you can use the Band Pass filter to create telephone or vintage phonograph sounds. With higher resonance settings, it can create buzzy or nasal timbres.
R (Band Reject). This filter type–also called a notch filter– cuts only the parts of the sound directly around the cutoff frequency. Try modulating the cutoff with an LFO to create phaser-like effects.
2-pole LP, HP, BP, and BR: These produce 12dB/octave slopes for LP and HP, and 6dB/octave for BP and BR. A classic synthesizer expander module used this type of filter.
4-pole LP, HP, BP, and BR: These produce 24dB/octave slopes for LFP and HP, and 12dB/octave for BP and BR. In comparison to 2-pole filters, these create a sharper roll-off beyond the cutoff frequency with more delicate resonance.
Many classic analog synths used this general type of filter.
Multi Filter. This is a complex filter which is capable of all of the 2-pole filter types, and many more besides. See “Multi Filter,” on page 43.

滤波器类型 (FILTER TYPE)
[ Polysix、MS-20 LP、MS-20 HP、2 极 LP、2 极 HP、2 极 BP、2 极 BR、4 极 LP、4 极 HP、4 极 BP、4 极 BR、多重滤波器 ] 按下前面板 FILTER TYPE 按钮可选择和编辑类型参数。可直接使用三种有用的设置：Polysix LP、MS-20 LP 和 2 极谐振 HP。通过编辑屏幕参数，可获得更多类型。如果在这样做时选择前面板上列出的三种类型之一，前面板 FILTER TYPE LED 将更新以显示选择。选择任何其他滤波器类型都会点亮 MORE LED。如果从前面板按钮在类型之间切换，MORE 位置会记住最后使用的“更多”类型。
LP（低通滤波器）。这会切断声音中高于截止频率的部分。低通滤波器是最常见的滤波器类型，用于使明亮的音色听起来更暗淡。
HP（高通滤波器）。这会切断声音中低于截止频率的部分。您可以使用它使音色听起来更薄或更嗡嗡作响。
BP（带通滤波器）。这会切断声音的所有部分，包括高音和低音，但截止频率周围的区域除外。由于此滤波器会切断高频和低频，因此其效果会根据截止设置和振荡器的多重采样而发生巨大变化。
使用低共振设置，您可以使用带通滤波器来创建电话或老式留声机的声音。使用较高的共振设置，它可以创建嗡嗡声或鼻音。
R（带阻滤波器）。这种滤波器类型（也称为陷波滤波器）仅切断声音中直接位于截止频率周围的部分。尝试使用 LFO 调制截止频率以创建类似移相器的效果。
2 极 LP、HP、BP 和 BR：这些滤波器为 LP 和 HP 产生 12dB/八度斜率，为 BP 和 BR 产生 6dB/八度斜率。经典合成器扩展器模块使用这种类型的滤波器。
4 极 LP、HP、BP 和 BR：这些滤波器为 LFP 和 HP 产生 24dB/八度斜率，为 BP 和 BR 产生 12dB/八度斜率。与 2 极滤波器相比，它们在截止频率之外产生更尖锐的滚降，共振更细腻。
许多经典模拟合成器使用这种通用类型的滤波器。
多重滤波器。这是一个复杂的滤波器，能够实现所有 2 极滤波器类型，以及更多其他类型。请参阅第 43 页的“多重滤波器”。

MS-20 LP and HP: These 12dB/octave, self-resonating filters lovingly recreate the distinctive timbral signature of the classic Korg MS-20. Raising Resonance will cause increasing amounts of saturation and overdrive, creating a more aggressive tone. Input volume can have a strong effect on this character; for more information, see “Gain” on page 42.
Polysix: This 24dB/octave, self-resonating low-pass filter provides the strong, sweet sound of the classic Korg Polysix.
MS-20 LP 和 HP：这些 12dB/八度、自谐振滤波器精心重现了经典 Korg MS-20 的独特音色特征。提高谐振会导致饱和度和过载量增加，从而产生更具侵略性的音调。输入音量可以对此特性产生强烈影响；有关更多信息，请参阅第 42 页的“增益”。
Polysix：这个 24dB/八度、自谐振低通滤波器提供了经典 Korg Polysix 强劲、甜美的声音。

What’s a Multi Filter?
Standard multimode filters generate low-pass, high-pass, and band-pass filters simultaneously—but only let you use one of them at a time. The Multi Filter gives you access to all three filter modes simultaneously, in any combination, along with the dry input signal. You can choose from a large number of preset combinations, or create your own complex filter modes using the Manual controls.
This is capable of some cool sounds in and of itself, but things really get interesting when you use Crossfade (see “Xfd (Crossfade)” on page 43). Crossfade allows you to mix between two of these filter settings (Mode 1 and Mode 2), using modulation sources such as envelopes, LFOs, or real-time controllers.
什么是多重滤波器？
标准多模式滤波器可同时生成低通、高通和带通滤波器，但每次只能使用其中一种。多重滤波器可让您同时以任意组合访问所有三种滤波器模式以及干输入信号。您可以从大量预设组合中进行选择，也可以使用手动控件创建自己的复杂滤波器模式。
这本身就能产生一些很酷的声音，但当您使用 Crossfade 时，事情会变得非常有趣（请参阅第 43 页的“Xfd（Crossfade）”）。Crossfade 允许您使用调制源（例如包络、LFO 或实时控制器）在两种滤波器设置（模式 1 和模式 2）之间进行混合。

				Filter Key Track
Keyboard Track
Most acoustic instruments get brighter as you play higher pitches. At its most basic, keyboard tracking re-creates this effect by increasing the cutoff frequency of a lowpass filter as you play higher on the keyboard. Usually, some amount of key tracking is necessary in order to make the timbre consistent across the entire range.

The wavestate keyboard tracking can also be much more complex, since it allows you to create different rates of change over up to four different parts of the keyboard. For instance, you can: • Make the filter cutoff increase very quickly over the middle of the keyboard, and then open more slowly–or not at all–in the higher octaves.

• Make the cutoff increase as you play lower on the keyboard.

• Create abrupt changes at certain keys, for split-like effects.
键盘音轨
大多数原声乐器在弹奏高音时会变得更明亮。最基本的键盘跟踪是通过在键盘上弹奏更高音时增加低通滤波器的截止频率来重现这种效果。通常，需要一定量的键跟踪才能使音色在整个范围内保持一致。

波形键盘跟踪也可能更加复杂，因为它允许您在键盘的最多四个不同部分上创建不同的变化率。例如，您可以：• 使滤波器截止频率在键盘中间非常快速地增加，然后在高八度音阶中更慢地打开 - 或根本不打开。

• 使截止频率在键盘上弹奏较低音时增加。

• 在某些键上创建突然变化，以获得类似分割的效果。

How Key Track works: Keys and Slopes
The keyboard tracking works by creating four ramps, or slopes, between five keys on the keyboard. The bottom and top keys are fixed at the bottom and top of the MIDI range, respectively. You can set the other three keys–named Low, Mid, and High–to be anywhere in between.

The four Slope values control the rate of change between each pair of keys. For instance, if the Low-Mid Slope is set to 0, the value will stay the same between the Low Key and the Mid Key.

You can think of the resulting shape as being like two folding doors attached to a hinge in the center. At the Mid Key (the main hinge), the keyboard tracking has no effect.

The two folding doors swing out from this center point to create changes in the higher and lower ranges of the keyboard.
键轨的工作原理：键和斜率
键盘跟踪的工作原理是在键盘上的五个键之间创建四个斜坡或斜率。底部和顶部的键分别固定在 MIDI 范围的底部和顶部。您可以将其他三个键（称为低、中和高）设置为介于两者之间的任何位置。

四个斜率值控制每对键之间的变化率。例如，如果低中斜率设置为 0，则低键和中键之间的值将保持不变。

您可以将生成的形状想象成连接到中心铰链的两扇折叠门。在中键（主铰链）处，键盘跟踪不起作用。

两扇折叠门从这个中心点向外摆动，以在键盘的较高和较低范围内产生变化。

Slope
Positive slope values mean that the keyboard tracking output increases as you play farther from the Mid Key; negative slope values mean that it decreases. Because of this, the meaning of positive and negative slopes changes depending on whether the slope is to the left or right of the Mid Key.
Low and Low-Mid: negative slopes make the output go down as you play lower on the keyboard, and positive slopes make the output go up.
Mid-High and High: negative slopes make the output go down as you play higher on the keyboard, and positive slopes make the output go up.

[Constant Rate, Constant Time]

Amp
Notes on Amp Modulation
Amp modulation scales the Amp Level and Amp Envelope, with an upper limit of 2x the original settings to prevent uncontrollably loud output. If the original level is low, the maximum modulated volume will also be low.
(For the technically inclined: Amp modulation is multiplicative. Modulation values are offset by +1.0, so that a value of 0 has no effect. The maximum modulation value—for example, moving the Mod Wheel all the way up—doubles the volume level if the Intensity is +100%, or brings the level to silence if the Intensity is -100%.)
Tip: To start with a low level (or silence) and then use modulation to raise the level to maximum, use the Filter Output Level instead.	
放大器
放大器调制注意事项
放大器调制会调整放大器电平和放大器包络，上限为原始设置的 2 倍，以防止输出不受控制的响度。如果原始电平较低，则最大调制音量也会较低。
（对于技术倾向者：放大器调制是乘法。调制值偏移 +1.0，因此 0 值没有影响。最大调制值（例如，将调制轮一直向上移动）在强度为 +100% 时使音量加倍，在强度为 -100% 时使音量降至静音。）

Using Modulation
Modulation Overview
Most front-panel controls and on-screen parameters can be modulated. You can also create multiple modulation routings to the same destination. Each modulation routing includes a primary modulation source, an intensity, and a secondary modulation source; the three are multiplied together to create the modulation amount. Once a modulation routing is created, you can change either of the sources, but you cannot change the destination.
使用调制
调制概述
大多数前面板控件和屏幕参数都可以进行调制。您还可以创建到同一目标的多个调制路由。每个调制路由都包括一个主调制源、一个强度和一个次调制源；三者相乘以产生调制量。创建调制路由后，您可以更改任一源，但不能更改目标。

Adding Modulation Routings
To add a new Modulation routing:
1. In the display, select the parameter that you’d like to modulate. (If you want to modulate a knob, you can skip this step.)
2. Hold MOD and press >.
The Add New Modulation popup appears.
3. To select the modulation destination, move its front-panel control (e.g., CUTOFF), or press ENTER to select the parameter from step 1.
Note: If the selected parameter cannot be modulated, a message will appear on the display and then the popup will close.
4. To select the modulation source, move a controller (e.g. WHEEL 1) or a MOD KNOB, play a note (for Velocity), press a button for one of the LFOs, Envelopes, or the Step Sequencer, press WSEQ STEPS (for Step Pulse), or send a MIDI CC.
Alternatively, if you want to use a mod source which can’t be selected from the front panel (such as Exponential Velocity or Mod Processors), press ENTER and manually select the mod source in the following screen.
If the message “Chan Src Only” appears, the parameter cannot be modulated from per-voice sources such as LFOs, envelopes, the Step Seq Lane, and Key Track. (The Vector Env can be used, since there is a separate Performance-level version of this envelope.) Choose a different modulation source.
5. Press ENTER to create the mod routing, or SHIFT-ENTER to cancel.
The Mods page appears, showing the new routing.
6. Set the Intensity as desired.
The maximum Intensity is typically +/- the full range of the parameter, so that regardless of the programmed value, modulation can always reach the minimum or maximum values.
7. Optionally, assign a second modulator as the Int Mod Source (Intensity Mod Source), whose value multiplies that of the main Source.
For example, you could route the Step Sequencer to Multi Filter Crossfade, with the overall amount of the step sequencer modulated by the Pan LFO.

			Filter/Amp/Pitch Env Curve
Often, envelopes are pictured as being made out of straight lines. In actuality, they are more likely to be made out of curves.

In other words, each segment’s level will change quickly at first, and then slow down as it approaches the next point.

This tends to sound better than straight, linear segments.

Classic analog synth envelopes made these curved shapes naturally. The wavestate goes a step further than vintage synths, however, and lets you control the amount of curvature separately for each envelope segment.

When you change the curvature, the envelope times remain the same. However, greater curvatures will tend to sound faster, because the value changes more quickly at the beginning.
通常，人们将包络描绘成由直线构成。实际上，它们更可能是由曲线构成。

换句话说，每个段的电平一开始会快速变化，然后在接近下一个点时变慢。

这听起来往往比直线段更好。

经典的模拟合成器包络自然地形成了这些弯曲的形状。然而，波形比老式合成器更进一步，让您可以分别控制每个包络段的曲率。

当您改变曲率时，包络时间保持不变。但是，曲率越大，听起来就越快，因为值在开始时变化得越快。

Different curve settings for up and down Generally, upwards segments use different curvature settings than downward segments. For instance, a curve of 3 is a good default setting for upward segments, such as Attack. On the other hand, a curve of 6 or more is good for downward segments, such as Decay and Release.
上下曲线设置不同 通常，向上部分使用与向下部分不同的曲率设置。例如，对于向上部分（如 Attack），曲线 3 是良好的默认设置。另一方面，对于向下部分（如 Decay 和 Release），曲线 6 或更高是良好的设置。

These set the curvatures of the Attack, Decay, and Release segments, respectively. Hold SHIFT and move the ATTACK, DECAY, or RELEASE knobs to edit these parameters directly.

Trigger Threshold
[–100%…+100%]
This sets the modulation level which will trigger the envelope. Among other things, you can use this to adjust the exact point in an LFO’s phase at which the envelope will be reset, effectively controlling its “groove” against other rhythmic effects.
When the threshold is positive (or 0), the envelope resets when the Trigger Source value passes through the threshold moving upwards. In other words, if the value has been below the Threshold, and then changes so that it is equal to or greater than the Threshold, the envelope resets.
When the threshold is negative, the envelope resets when the Trigger Source value passes through the threshold moving downwards. In other words, if the value has been above the Threshold, and then changes so that it is equal to or below the Threshold, the envelope resets.

Most of the waveforms are self-explanatory, but a few will benefit from more details:

Rndm° (Random Start Phase)
[Off, On]
When Random Start Phase is On, the LFO will start at a different, random phase every time. If one of the Random waveforms is selected, this parameter is not shown.
Offset
[–100…+100]
By default, almost all of the LFO waveforms are centered around 0, and then swing all the way from -100 to +100.
Offset lets you shift the LFO up and down, so that—for instance—it’s centered on 50, and then swings from -50 to +150.
For example, let’s say that you’re using an LFO for vibrato. If Offset is 0, the vibrato will be centered on the note’s original pitch, bending it both up and down.
If Offset is +100, on the other hand, the vibrato will only raise the pitch above the original note.
开始相位
[-180…+180]
这控制 LFO 启动时波形的相位（请参阅第 67 页上的“延迟、淡入淡出、开始相位和随机相位何时应用？”）。如果“随机开始相位”处于开启状态，或者选择了随机波形之一，则不会显示此参数。
Rndm°（随机开始相位）
[关闭，开启]

偏移
[–100…+100]
默认情况下，几乎所有 LFO 波形都以 0 为中心，然后从 -100 一直摆动到 +100。
偏移允许您上下移动 LFO，例如，它以 50 为中心，然后从 -50 摆动到 +150。
例如，假设您使用 LFO 进行颤音。如果 Offset 为 0，颤音将以音符的原始音高为中心，上下弯曲。
另一方面，如果 Offset 为 +100，颤音只会将音高提高到原始音符之上。

Curve (CURVE)
[–100…+100]
Curve adds curvature to the basic waveform. As you can see in the graphic below, this can make the waveforms either more rounded or more extreme. It can also be useful to emphasize certain value ranges, and deemphasize others.

Sync Notes Mode
[Sync Notes, Free Run]
This controls how the LFO behaves when SYNC NOTES is enabled.
Sync Notes: Each note has its own LFO. If no other notes are sounding, the LFO starts as it would normally. If other notes are sounding, the LFO starts by matching the phase of the oldest sounding note. The result is that the LFOs for all of the notes are initially synchronized together. This option is generally the most effective for synchronizing with tempo and Motion Sequences.
Note: With Sync Notes, each note’s LFO speed may still be different if modulated by note number, velocity, key scaling, or other note-specific modulation sources.
Free Run: All notes in the Program share a single LFO. This is particularly useful if you want a slow LFO shape to continue running under multiple phrases, or under an arpeggiator pattern. The LFO starts running as soon as you select the Program or Performance, before any notes are played.
滤波器/放大器/音高/声像 LFO 详细信息
同步音符模式
[同步音符，自由运行]
这控制启用 SYNC NOTES 时 LFO 的行为方式。
同步音符：每个音符都有自己的 LFO。如果没有其他音符发声，LFO 将按正常方式启动。如果有其他音符发声，LFO 将通过匹配最早发声音符的相位来启动。结果是所有音符的 LFO 最初都同步在一起。此选项通常是与节奏和运动序列同步的最有效方法。
注意：使用同步音符时，如果通过音符编号、速度、键缩放或其他特定于音符的调制源进行调制，每个音符的 LFO 速度可能仍然不同。
自由运行：程序中的所有音符共享一个 LFO。如果您希望慢速 LFO 形状在多个乐句或琶音器模式下继续运行，这将特别有用。只要您选择程序或性能，LFO 就会开始运行，在播放任何音符之前。

Trigger Src (Trigger Source)
[List of modulation sources]
This selects a source to re-start the LFO from the Start Phase. For example, you can insert a MIDI CC trigger into a MIDI sequence so that a free-running LFO sounds the same every time the sequence is played.
You can also use Gate + Damper to reset the LFO at the start of phrases played on the keyboard. Note that this may not behave exactly the same as Sync Notes.
触发器源 (触发器源)
[调制源列表]
这将选择一个源，从起始阶段重新启动 LFO。例如，您可以将 MIDI CC 触发器插入 MIDI 序列，这样每次播放序列时，自由运行的 LFO 听起来都一样。
您还可以使用 Gate + Damper 在键盘上弹奏的乐句开始时重置 LFO。请注意，这可能与同步音符的行为不完全相同。

Trigger Threshold
[–100%…+100%]
This sets the modulation level which will trigger the LFO.
When the threshold is positive (or 0), the LFO resets when the value passes through the threshold moving upwards. In other words, if the value has been below the Threshold, and then changes so that it is equal to or greater than the Threshold, the LFO resets.
When the threshold is negative, the LFO resets when the value passes through the threshold moving downwards. In other words, if the value has been above the Threshold, and then changes so that it is equal to or below the Threshold, the LFO resets.
Note: modulation sources may not always reliably reach the extreme values of +100 or -100. This can happen due to smoothing, or with LFOs when using certain shapes and faster speeds. If this is the case, Thresholds of +100 or -100 will cause inconsistent behavior. To avoid this issue, reduce the Threshold until the LFO resets consistently.
触发阈值
[–100%…+100%]
这将设置触发 LFO 的调制级别。
当阈值为正（或 0）时，当值向上移动通过阈值时，LFO 会重置。换句话说，如果值低于阈值，然后发生变化，使其等于或大于阈值，则 LFO 会重置。
当阈值为负时，当值向下移动通过阈值时，LFO 会重置。换句话说，如果值高于阈值，然后发生变化，使其等于或低于阈值，则 LFO 会重置。
注意：调制源可能并不总是可靠地达到 +100 或 -100 的极值。这可能是由于平滑，或者在使用某些形状和更快速度时使用 LFO 而发生的。如果是这种情况，+100 或 -100 的阈值将导致不一致的行为。为了避免此问题，请降低阈值直到 LFO 持续重置。

				Modulation Processors
Overview
Modulation Processors transform a modulation signal to make it into something new. The original modulation signal also remains available. There are two Modulation Processors per Program.
The Modulation Processor outputs appear in the list of modulation sources, just like the LFOs and Envelopes.
Type
[Gate, Offset, Quantize, Scale, Curve, Smooth, Sum ]
This controls the type of processing performed by the Mod Processor. Each is described in detail below.
概述
调制处理器将调制信号转换为新信号。原始调制信号仍然可用。每个程序有两个调制处理器。
调制处理器输出出现在调制源列表中，就像 LFO 和包络一样。
类型
[门限、偏移、量化、缩放、曲线、平滑、总和]
这控制调制处理器执行的处理类型。下面详细介绍了每种处理类型。

Gate		
This lets you switch between two modulation sources (or fixed values) using a third modulation source.
It’s similar to an audio gate with a side-chain, but with even more flexibility–since you get to choose what happens when the gate is closed (below the threshold), as well as when it’s open (above the threshold).
For instance, you can use Gate to: 
• Apply pitch-bend or other effects to some notes, but not to others (using Control at Note-On Only) 
• Apply modulation only after the source reaches a certain threshold–for instance, use Velocity to modulate an LFO’s frequency, but only once Velocity is greater than 90 
• Use a controller to switch between two different LFOs (or any two modulation sources)
门限
这可让您使用第三个调制源在两个调制源（或固定值）之间切换。
它类似于带有侧链的音频门限，但灵活性更高 - 因为您可以选择当门限关闭（低于阈值）时以及打开（高于阈值）时发生的情况。
例如，您可以使用门限来：
• 对某些音符应用弯音或其他效果，但不对其他音符应用（仅在音符开启时使用控制）
• 仅在源达到特定阈值后应用调制 - 例如，使用速度调制 LFO 的频率，但仅在速度大于 90 时才应用
• 使用控制器在两个不同的 LFO（或任何两个调制源）之间切换

[List of Modulation Sources]
This selects the modulation source to control the Gate.

[-100…+100]
This sets the value of the Control Source at which the gate opens or closes.

The selected output will then remain active throughout the duration of the note, regardless of any subsequent change in the Control Source’s value. Note that the output value itself can continue to change; only the selection of Below or Above is fixed.

Converting from bipolar to unipolar
You can use Offset to convert a bipolar modulation source (both negative and positive), such as an LFO, to a unipolar signal (positive only). To do so:
1. Select the LFO as the source.
2. Set the Input Amount to 50%.
This cuts the overall level of the LFO in half.
3. Set the Offset to 50%.
This shifts the LFO signal up, so that it now swings between 0 and +100.
从双极转换为单极
您可以使用偏移将双极调制源（正负均可），例如 LFO，转换为单极信号（仅正极）。操作方法：
1. 选择 LFO 作为源。
2. 将输入量设置为 50%。
这会将 LFO 的整体电平降低一半。
3. 将偏移设置为 50%。
这会将 LFO 信号上移，使其现在在 0 和 +100 之间摆动。

Converting from unipolar to bipolar
Similarly, you can convert a unipolar modulation source to a bipolar signal:
1. Select the modulation source as desired.
2. Set the Input Amount to +200%.
This doubles the overall level of the modulation source.
3. Set the Offset to -100%.
This shifts the Modulation signal down, so that it ranges from -100 to +100.
从单极转换为双极
类似地，您可以将单极调制源转换为双极信号：
1. 根据需要选择调制源。
2. 将输入量设置为 +200%。
这会使调制源的整体电平加倍。
3. 将偏移设置为 -100%。
这会将调制信号向下移动，使其范围从 -100 到 +100。

Quantize
This changes a continuous signal into a series of discrete steps. Use this to change the shape of LFOs or envelopes, or to force a controller to land on a few specific values.
Input
[List of Modulation Sources]
This selects the input for the Mod Processor.
Steps
[1…32]
This controls the severity of the effect. Lower values create more “steppy” effects.
For instance, when this is set to 2, there will be “steps” at 0, 50, and 100. With a bipolar source, there will also be steps at –50 and –100.
量化
这会将连续信号转换为一系列离散步骤。使用它来更改 LFO 或包络的形状，或强制控制器落在几个特定值上。
输入
[调制源列表]
这会选择 Mod 处理器的输入。
步骤
[1…32]
这控制效果的严重程度。较低的值会产生更多“阶梯式”效果。
例如，当设置为 2 时，将在 0、50 和 100 处有“步骤”。对于双极源，在 -50 和 -100 处也会有步骤。

Quantized Pitch Bend
You can use Quantize to create quantized pitch bend, for fret-dragging effects, brass rips, and more.
1. Open the Add New Modulation popup (MOD + >), and set the Destination to Pitch Tune.
2. Press ENTER to close Add New Modulation and bring up the Modulation Source popup (since there isn’t a front-panel control to select for Mod Processors).
3. Press the PAN LFO button to jump to the middle of the Generators section, and then scroll down to select Mod Processor 1. Press ENTER to confirm.
4. Set the Intensity to any exact half-step value: +5.00, +7.00, etc.
5. Set Mod Processor 1’s Type to Quantize, and set the Input to Pitch Bend.
6. Set Steps to the same number you used for Intensity, above.
7. On the Pitch Mod page, set both Pitch Bend Up and Pitch Bend Down to 0 (so that only the Mod Processor version will affect Pitch).
Now, moving the Pitch Bend wheel will create quantized pitch bends.
量化弯音
您可以使用量化来创建量化弯音，以实现音品拖拽效果、铜管音色转换等。
1. 打开“添加新调制”弹出窗口 (MOD + >)，并将目标设置为音高调谐。
2. 按 ENTER 关闭“添加新调制”并调出“调制源”弹出窗口（因为没有前面板控件可以选择调制处理器）。
3. 按 PAN LFO 按钮跳转到生成器部分的中间，然后向下滚动以选择调制处理器 1。按 ENTER 确认。
4. 将强度设置为任何精确的半步值：+5.00、+7.00 等。
5. 将调制处理器 1 的类型设置为量化，并将输入设置为弯音。
6. 将步数设置为与上面强度相同的数字。
7. 在 Pitch Mod 页面上，将 Pitch Bend Up 和 Pitch Bend Down 都设置为 0（这样只有 Mod Processor 版本会影响音高）。
现在，移动 Pitch Bend 轮将创建量化的音高弯音。

Bipolar and Unipolar modulation sources To understand Curve, it helps to understand the difference between bipolar and unipolar modulation sources.

Bipolar sources can swing all the way from –100 to +100, with 0 in the middle. Most LFOs are bipolar, for instance; so is Pitch Bend. Generally, bipolar modulation sources will work better with the Asymmetric mode, but Symmetric may also produce interesting results.

Unipolar sources only go from 0 to 100, with 50 in the middle. Often, MIDI controllers are unipolar (though you can also select bipolar versions). Envelopes are usually programmed to be unipolar, even though the Filter and Pitch Envelopes allow negative levels.

With unipolar sources, it’s almost always better to use the Symmetric mode; Asymmetric can cause offsets and other strange results.
双极和单极调制源 要理解曲线，了解双极和单极调制源之间的差异会有所帮助。

双极源可以从 -100 一直摆动到 +100，中间为 0。例如，大多数 LFO 都是双极的；弯音也是如此。通常，双极调制源在非对称模式下效果会更好，但对称也可能产生有趣的结果。

单极源仅从 0 到 100，中间为 50。通常，MIDI 控制器是单极的（尽管您也可以选择双极版本）。包络通常被编程为单极，即使滤波器和音高包络允许负电平。

对于单极源，使用对称模式几乎总是更好；非对称可能会导致偏移和其他奇怪的结果。

下面介绍了一些出厂音阶。
阿拉伯语：包括阿拉伯音乐中使用的四分之一音程。
			
Pythagoras: Especially effective for melodies, this produces 11 completely pure fifths, at the expense of other intervals—thirds in particular. To maintain the tuning of the octave, the final fifth—from sharp four to sharp root—is also quite flat.
Werkmeister (Werkmeister III), Kirnberger (Kirnberger III): These are “Well-Tempered” tunings from the Baroque period.
Pelog: This Indonesian gamelan scale uses seven notes per octave. When Key is set to C, use the white keys. The black keys will play the equal tempered pitches.

Slendro: This Indonesian gamelan scale has five notes per octave. When Key is set to C, use C, D, F, G and A. Other notes will play the normal equal-tempered pitches.
Stretch: In this acoustic piano tuning, notes below the middle of the keyboard are progressively more flat, and notes above the middle are progressively more sharp.
毕达哥拉斯：这对旋律特别有效，它会产生 11 个完全纯五度，而牺牲其他音程（尤其是三度）。为了保持八度的调音，最后的五度（从升四度到升根音）也相当平坦。
Werkmeister（Werkmeister III）、Kirnberger（Kirnberger III）：这些是巴洛克时期的“平均律”调音。
Pelog：这种印度尼西亚甘美兰音阶每八度使用七个音符。当调设置为 C 时，使用白键。黑键将演奏平均律音高。
纯大调、纯小调：这些是分别针对大调和小调优化的纯律。与其他音阶不同，它们保持由全局调音参数设置的 A4 调音（例如 A=440 Hz）（请参阅第 83 页的“调音”）。
因此，所选调的根音可能会偏离其平均律音高。
Slendro：这种印尼甘美兰音阶每八度有五个音符。当调设置为 C 时，使用 C、D、F、G 和 A。其他音符将演奏正常的十二平均律音高。
拉伸：在这种钢琴调音中，键盘中间以下的音符逐渐变平，中间以上的音符逐渐变尖。

Transpose
[Transposes Scale, Within Scale]
This parameter determines how the Performance and Program Transpose parameters interact with the Scale and Key.
Within Scale means that transposition uses intervals within the current scale. This option works well if you are using non-equal-tempered scales and want to build parallel intervals (such as “fat fifths”) into your sounds. For example, let’s say that you are using Just Intonation in the key of C. If you transpose one of the Programs up by 7 semitones and play a C, the result will be two notes: C, and G 2 cents sharp (the Just Intonation fifth).
Transposes Scale means that the Scale’s Key changes to match the transposition. This option is useful if you play in non-equal-tempered scales and transpose entire songs—to accommodate singers, for example. In other words, let’s say that the Key is set to C, and the third note of the scale (E) is 50 cents flat. If you transpose up by 2 semitones, and play an E, the result will be an F# 50 cents flat—still the third note of the scale. For this application, it’s also best to set the Convert Position to Post-MIDI; for more information, see “Chan+Poly: The mod source “Aftertouch” receives both Channel and Poly Aftertouch. If both are sent simultaneously, the most recent value is used.” on page 86.

Auto Power-Off
[Disabled, 4 Hours]
By default, to conserve energy, the wavestate will automatically turn off after about four hours have elapsed without use of the front panel, keyboard, or MIDI input, and without being connected to the Editor/Librarian. You can disable this feature, if desired.
When the power turns off, any un-saved edits or settings will be lost. Make sure that you save important settings before this occurs.
